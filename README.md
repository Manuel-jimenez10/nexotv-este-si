# NEXOTV

1. Clonar el proyecto

2. Copiar y renombrar el archivo **.env.template** a **.env**

3. Ejecutar

```
yarn install
```

4. Levantar la imagen de Docker

```
docker-compose up -d
```

5. Levantar el servidor

```
yarn start:dev
```

6. Visitar el sitio

```
localhost:3000/graphql
```
